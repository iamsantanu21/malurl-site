How it works

  1. 1. Install the extension using the buttons above.
  2. 2. Open any website or paste a URL into the MalURL popup.
  3. 3. The URL is sent to a FastAPI server hosted on Render, where the LightGBM model runs.
  4. 4. You immediately see the prediction and probability for each class.
